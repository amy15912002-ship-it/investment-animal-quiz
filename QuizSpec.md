### 2.1 封面頁

- 顯示測驗標題「揭曉！你是哪隻阿牛？」
- 副標「阿牛今天想嘗試不同的投資風格——哪套動物裝最適合你？」
- 動物裝預覽：展示部分動物剪影（製造好奇心）
- CTA 按鈕「開始測驗」
- 底部提示「10 道生活題 · 約 1.5 分鐘 · 純娛樂性質」

### 2.2 題目頁

- 頂部進度條：顯示「N / 10」及百分比
- 題目標題置中顯示
- 4 個選項卡片（icon + 文字），點擊後高亮
- 選擇後自動跳轉下一題（400ms 延遲）
- 最後一題選完後自動進入計算頁
- 左上角「上一題」按鈕（第 1 題時隱藏）

### 2.3 計算頁

- 顯示「阿牛正在找最適合你的動物裝...」
- 三點跳動動畫
- 依序切換提示文字（0.6 秒間隔）：
  1. 分析你的投資座標中...
  2. 比對風險偏好...
  3. 測量主動程度...
  4. 就快揭曉了！
- 總停留 2.5 秒後自動跳轉結果頁

### 2.4 結果頁

由上至下依序為：

1. **主動物頭像**（阿牛穿動物裝插圖 + 彩色圓底）
2. **動物名稱 + 系列名**
3. **語錄引言**（虛線框）
4. **性格描述**（約 50 字）
5. **投資超能力**卡片（品牌主色底）
6. **適合的投資方式**卡片（次色底）
7. **主推產品推薦**卡片（詳見 §6）
8. **性格標籤**（6 個 tag pill）
9. **投資動物相性**（最佳戰友 / 最佳搭檔 / 最佳旅伴）
10. **撒花動畫**（confetti）
11. **底部操作**：「再玩一次」+ 社群分享

## 4. 計分機制

### 4.1 雙維度座標系統

每個選項對應兩個維度的座標位移值：

```
維度一（X 軸）：主動/委外軸
  +2 = 完全主動掌控（自己研究、自己決策）
   0 = 中性
  -2 = 完全委外自動（交給系統/專家）

維度二（Y 軸）：風險偏好軸
  +2 = 積極進取（高報酬、高波動）
   0 = 中性
  -2 = 穩健保守（低波動、保本優先）
```

### 4.2 計分流程

```javascript
// 初始化座標
position = { x: 0, y: 0 }

// 每題作答後，累加該選項的位移值
position.x += selectedOption.deltaX
position.y += selectedOption.deltaY

// 10 題全部作答完畢後，計算與各動物目標座標的歐氏距離
// 取最近者為主屬性，取第二近者為副屬性
```

### 4.3 動物目標座標

（以10題總分為尺度，X/Y 範圍約 -20 ~ +20）

| 動物 | 目標X（主動軸） | 目標Y（風險軸） | 說明 |
|------|--------------|--------------|------|
| 🦉 牛頭鷹 | +18 | +6 | 高度主動 × 中積極 |
| 🦁 阿牛獅 | -4 | +17 | 偏委外 × 高積極 |
| 🐘 大象牛 | -5 | -8 | 偏自動 × 穩健 |
| 🐆 衝豹牛 | +7 | +17 | 偏主動 × 高積極 |
| 🐝 蜜蜂牛 | -16 | 0 | 全自動 × 中性 |
| 🦝 浣熊牛 | -16 | -16 | 全自動 × 超保守 |

### 4.4 結果判定

```
1. 計算 position 與所有動物目標座標的歐氏距離
2. 距離最小者 = 主屬性動物（主推產品）
3. 同分/同距離：目標座標Y軸值較高者優先（風險偏好明顯者優先）
```

### 4.5 浣熊牛出現率說明

浣熊牛目標座標位於極端保守 × 全自動的角落（-16, -16）。用戶需要在多個問題中持續選擇「最保守」且「最委外」的選項，才能讓座標落在浣熊牛的吸引範圍內，因此出現頻率相對較低，但不設稀有特效。

## 6. 產品推薦機制

### 6.1 推薦邏輯（更新為四大產品）

| 產品 | 推薦給 | 核心邏輯 |
|------|--------|---------|
| **自由 Pay** | 主動掌控型（牛頭鷹） | 精算型人格，自訂提領比例與時機 |
| **強勢王** | 積極委外型（阿牛獅） | 大膽進取，交給投研團隊選強勢投組 |
| **超底王** | 自動紀律型（衝豹牛、大象牛） | 跌幅自動加碼，比親自盯盤更快更準 |
| **鉅寶盆** | 循環複利型（蜜蜂牛） | 母子基金自動循環，停利轉入不間斷 |
| **定期定額** | 超穩健型（浣熊牛） | 小額自動存入，最基礎的紀律投資 |

### 6.2 產品 × 動物完整對應

| 動物 | 主推產品 | 主推理由 |
|------|---------|---------|
| 🦉 牛頭鷹 | **自由 Pay** | 精算型人格，自訂提領比例與時機最能發揮其分析能力 |
| 🦁 阿牛獅 | **強勢王** | 積極進取，投研團隊選好強勢投組，一出手就是重點布局 |
| 🐘 大象牛 | **超底王** | 已有定期定額習慣，超底王在原有節奏上自動加碼攤低成本 |
| 🐆 衝豹牛 | **超底王** | 追求精準時機，跌幅觸發比親自盯盤更快更準 |
| 🐝 蜜蜂牛 | **鉅寶盆** | 停利自動轉入循環，讓錢幫錢賺錢，完全符合複利循環哲學 |
| 🦝 浣熊牛 | **定期定額** | 小額穩健自動存入，最適合謹慎保守者的入門工具 |

### 6.3 產品推薦文案

#### 🦉 牛頭鷹 → 自由 Pay

> 你擅長分析與策略規劃，自由 Pay 讓你自訂提領比例與時機，像排行程一樣精準掌控現金流。依比例或依金額，由你決定。
> *你的數據腦 + 自由 Pay 的彈性調整 = 完美搭配*

#### 🦁 阿牛獅 → 強勢王

> 你夠大膽，你需要的是更強的武器。強勢王由投研團隊幫你精選強勢投組，單筆 + 月月扣 + 強勢超底三管齊下，讓你的每一次進場都是重點布局。
> *王者不自己選基金——王者選最強的團隊*

#### 🐘 大象牛 → 超底王

> 你本來就在定期定額，超底王讓你在原有節奏上多一層保護——跌了自動加碼，不用改變習慣就能攤低成本，穩中帶進。
> *定期定額的自然進化版，為你量身打造*

#### 🐆 衝豹牛 → 超底王

> 你最怕錯過時機，超底王幫你盯盤——設好小跌、大跌兩段觸發門檻，跌到位時自動秒速出手，比你親自守在螢幕前更快更準。
> *兩段式跌幅觸發，永遠比你的手指更快*

#### 🐝 蜜蜂牛 → 鉅寶盆

> 你懂得讓系統幫你工作。鉅寶盆的母子基金循環機制就是為你設計的——母基金穩健打底，定期自動轉入子基金追成長，達到停利條件再自動轉回，周而復始，複利不間斷。
> *設定一次，讓錢幫你自動釀蜜*

#### 🦝 浣熊牛 → 定期定額

> 你是投資世界的穩健派。定期定額讓你每個月自動存入，不需要擇時、不需要盯盤，用時間和紀律慢慢累積屬於你的資產。
> *最簡單的投資方式，也是最不容易出錯的*

## 9. 頁面流程狀態機

```
State: COVER
  ├─ [點擊「開始測驗」] → QUESTION(0)

State: QUESTION(n)  // n = 0~9
  ├─ [選擇選項, n < 9] → answers[n] = selectedIndex, 400ms 後自動 → QUESTION(n+1)
  ├─ [選擇選項, n = 9] → answers[n] = selectedIndex, 400ms 後自動 → CALCULATING
  └─ [點擊「上一題」, n > 0] → QUESTION(n-1)

State: CALCULATING
  ├─ 顯示 2.5 秒動畫
  ├─ 每 0.6 秒切換提示文字（共 4 段）
  └─ [2.5 秒後自動] → RESULT

State: RESULT
  ├─ 計算雙維度座標 → 取最近動物（主）
  ├─ 渲染結果頁（主動物 + 主推產品）
  ├─ 觸發 confetti 動畫（浣熊牛：稀有特效加強版）
  ├─ [點擊「再玩一次」] → COVER → 自動進入 QUESTION(0)
  └─ [點擊分享按鈕] → 開啟對應社群平台分享
```
